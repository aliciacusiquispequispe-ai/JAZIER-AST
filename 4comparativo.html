4. Cuadro comparativo

Permite comparar conceptos, teorías o personajes en una tabla visual.

Es una tabla organizada que visualiza las semejanzas y diferencias entre dos o más elementos, objetos o conceptos para facilitar su comprensión y análisis


  1. Identificar los elementos a comparar:

    2. Elige los temas, productos u objetos que quieres comparar. Asegúrate de que sean relevantes para tu objetivo.

  2. Seleccionar los criterios de comparación:

    3. Define los aspectos o parámetros clave que se utilizarán para la comparación, como características, diferencias, similitudes, precios o funciones.

  3. Recopilar y resumir la información:

    4.

    Busca y reúne la información necesaria para cada elemento. Es crucial que la información sea concisa y esté resumida, evitando párrafos largos.

  4. Estructurar el cuadro:

    1. Crea una tabla con columnas y filas.
    2. Coloca los elementos a comparar en las columnas superiores.
    3. Coloca los criterios de comparación en las filas de la izquierda.

  5. Rellenar las celdas:

    6.

    Llena cada celda con la información correspondiente, asegurándote de que sea clara y consistente. Puedes usar colores o iconos para que sea más fácil de leer y entender.

  6. Analizar la información:

    7. Una vez completado, evalúa el cuadro para identificar las similitudes y diferencias más relevantes entre los elementos y tomar decisiones si es necesario.
